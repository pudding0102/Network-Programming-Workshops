# Python Network Programming for Modern Networks

This repository contains a series of lab exercises developed to explore network programming concepts using Python, ranging from basic socket communication to advanced adaptive and conceptual networking models.

## 📚 Course Overview
This project follows a weekly progression:
- Start with structured TCP communication
- Move into unreliable UDP environments
- Expand to distributed and decentralized systems
- End with adaptive and conceptual networking models

The goal is to understand not just how networks work, but how they behave under uncertainty.

---

## 🧪 Weekly Labs

### Week 1 – TCP Client–Server
Basic socket communication using request-response model.

### Week 2 – UDP Communication
Connectionless communication and packet-based messaging.

### Week 3 – Broadcast Communication
One-to-many communication within a LAN.

### Week 4 – Multicast Communication
Group-based communication with selective receivers.

### Week 5 – Peer-to-Peer Networking
Decentralized communication with symmetric nodes.

### Week 6 – Ad-Hoc Networking (MANET)
Dynamic neighbor-based communication and probabilistic forwarding.

### Week 7 – Store-and-Forward
Message queuing and retry mechanisms for unreliable networks.

### Week 8 – Opportunistic Routing
Forwarding decisions based on delivery probability.

### Week 9 – Bio-Inspired Networking
Adaptive routing using pheromone-based reinforcement.

### Week 10 – Quantum-Inspired Networking
Conceptual simulation of one-time-read messages and state collapse.
